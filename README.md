# Two Body Orbital Simulation
This simulation uses html to depict how an orbiting body reacts around different kinds of stars or celestial bodies. 
There are two main ways to play around: you can move the orbiting body yourself, drag it and simply release it. You can also release it
with a swipe to make it orbit around the star.
Or, there's the slingshot mode, which allows you to launch the orbiting body using a slingshot effect, and shows you the predicted trajectory.
There is an animation for collisions as well.
Several dust particles orbit around the star as well and are affected by the collision impact between the orbiting body and the star.
